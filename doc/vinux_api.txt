    it in the quickfix window.

te#utils#run_command({command}, [, {callback}] [,{arglist}] [, {flag}])         *te#utils#run_command*

    Run {command} in background. If {command} is a empty string, you can enter
    the command string in cmdline window.
    
    Other parameter:
    1. {callback} is a function type, it will be called once command finished.
    Optional.
    2. arglist is list type, it will be passed to {callback} once command finished. Optional.
    3. flag is integer type, it specifies whether open quickfix window after
    {command} finish or not. Optional.

    Example:
      "run git fetch command in background
      :call te#utils#run_command('git fetch --all')
      "run command from user input and open quickfix window after command finisfinished
      :call te#utils#run_command("", 1)
      "run make cscope command in background, it will generate cscope
      database, after finished, te#pg#add_cscope_out will be called with a:path argument.
      call te#utils#run_command('make O=. ARCH=arm SUBARCH=sunxi COMPILED_SOURCE=1 '.g:tagging_program.cur_val,
                               \ function('te#pg#add_cscope_out'),[a:path])

ai-api                                                    *ai-api*
=============================================================================

tte#ai#get_provider_url()                                 *te#ai#get_provider_url()*
    Return the base URL string for the current AI provider.
    This function looks up the provider name via te#ai#get_provider_name()
    and returns the corresponding URL from s:provider_url_mapping.


tte#ai#get_provider_name()                                 *te#ai#get_provider_name()*
    Return the name of the current AI provider as a string.
    Retrieves the value of the global variable g:ai_provider_name.cur_val.


tte#ai#get_api_key()                                       *te#ai#get_api_key()*
    Return the API key string for the selected provider.
    Reads the first line from the token file located at
    ~/.config/<provider_name>.token.


tte#ai#get_model_name()                                    *te#ai#get_model_name()*
    Return the name of the selected LLM model.
    Reads the value of g:ai_llm_model_name.cur_val.


tte#ai#get_llm_model_list()                               *te#ai#get_llm_model_list()*
    Return a list of available LLM model names.
    Reads ~/.config/llm_model.list and returns non‑empty entries.


tte#ai#edit_ai_config(type)                               *te#ai#edit_ai_config(type)*
    Open the configuration file for editing.
    type==2 opens ~/.config/llm_model.list, otherwise opens
    ~/.config/<provider_name>.token.
